# Explainable Network Verification via Localized Subspecification

User Study System

This is a front-end system designed for user studies, including experimental pages for Group A and Group B as well as a warm-up question page. It is intended to collect and analyze users' behavioral data during the completion of specific tasks.

index.html: Serves as the entry point for the entire user study, including study instructions and a user ID input section.

groupA: Stores all pages for Group A (subspec with Q3, Q5)

groupB: Stores all pages for Group B (subspec with Q2, Q4)
